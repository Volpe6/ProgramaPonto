# Programa de ponto

Este programa foi criado visando realizar um trabalho repetitivo e chato, porém essencial, gerenciar pontos. 

Este programa abre uma tela bem simples, ela consiste em um label e dois botões. Um dos botões assume duas funções diferentes que são: iniciar e finalizar, ambos fazem a mesma coisa pegam a hora atual do computador no momento que foi clicado, o outro botão é o salvar, ele salva os dados obtidos com botões anteriores em um arquivo. No label aparece uma informação referente ao clique de algum botão, por exemplo quando clicado no botão salvar, é mostrado o local onde o arquivo foi salvo.

Como o programa funciona, ele abre um arquivo e o altera, o botão iniciar adiciona a hora que foi clicado no arquivo na coluna hora de entrada, e o finalizar faz a mesma coisa na coluna hora de saida.

Os dados necessários do usuário são: nome completo, modalidade da bolsa, e observações caso haja alguma. Estes dados devem ser informados no arquivo tela.py dentro da pasta src.
